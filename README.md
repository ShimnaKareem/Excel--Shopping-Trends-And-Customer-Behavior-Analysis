# Excel--Shopping-Trends-And-Customer-Behavior-Analysis
Excel dashboard and  sales analysis project
This Exceldashboard provide insights into customer shopping behavior and sales performance.The dashboard includes:
* Total Revenue Analysis
* Average Purchase Value
* Total orders overview
* promo code usage Breakdown
* Top High-Spending Locations
* Payment method sales trends
* season-wise sales Trends
* Purchase Frequency Analysis
* Interactive filters for Season and Category

TOOLS USED:
*Microsoft excel
*Pivot Tables
*Pivot Charts
*Slicers
*KPIs
*power query(Data cleaning)
*Dashboard Design &Data Visualization

The objective of this project is to identify customer purchasung patterns and support data-driven business decisions.
